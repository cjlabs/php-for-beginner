# String

---

### Menggabungkan String

```
<?php
$greeting = "Hello";
$name = "Putera Kahfi";
# menggabungkan string menggunakan dot (.)
echo $greeting." ".$name;
```



### Mengetahui jumlah karakter

```
<?php
$greeting " Hello Putera Kahfi";
$stringLength = strlen($greeting);
echo $stringLength;
```
